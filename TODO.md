## 🚀 次期マイルストーン: MVP (Minimum Viable Product)

仕様書で定義されたMVP機能の実装を目指します。

### P0: 必須機能

- [ ] **ノード追加:** ファイルとフォルダを新規に作成する (`Cmd+N`)
- [x] **ノード名変更:** アウトライン上で直接リネームする (`Enter`)
- [ ] **ノード削除:** ファイルとフォルダを削除する (`Delete`)
- [ ] **双方向同期の安定化:** ファイルシステム側での変更（移動、作成、削除、リネーム）を検知し、UIにリアルタイムで反映させる。

### P1: 重要機能

- [ ] **Markdown展開:** Markdownファイルの内容をアウトライン内で直接展開・表示する。
- [ ] **UI/UX改善:**
    - モダンなルックアンドフィールへのスタイリング調整。
    - 操作の成功・失敗を通知するUIフィードバック。
    - 大規模なフォルダ読み込み時のローディング表示。
